Tasks

Task topics provide instructions for completing specific tasks. They typically contain:
  • Information immediately required by users to accomplish a single task.
  • Minimal information to explain concepts to understand the task.
  • Links to related information that can help with executing the task.

They do not provide conceptual descriptions of product features or explanations of how something works, nor do they provide detailed data that might be needed to complete the task. Use concept and reference topics to provide this type of information.

Task topics include the following categories of information:

  • Installation
  • Configuration
  • Maintenance
  • Removal and replacement
  • Performance monitoring
  • Structure
  • Task Elements
  • Writing Task Topics
