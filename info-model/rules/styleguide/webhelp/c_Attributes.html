Attributes

Attributes specify properties of the content that can be used to determine how the content should be processed. Company Name uses them to:
  • Support content referencing
  • Support translation and localization
  • Filter content
  • Flag content

Many attributes are mentioned in this model within the context of the elements on which they can be used. However, this section provides a summary of common attributes available within the Company Name Information Model and Authoring Guidelines that can be used on many different elements.
