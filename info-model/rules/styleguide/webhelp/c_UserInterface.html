User Interface Elements

Use these inline elements when referring to elements within the graphic user interface:

Inline element Description Example code Output
user interface control <uicontrol> Use <uicontrol> to mark up any item that might appear on a screen, such as a window title, button label, menu option, or field name.    
file path <filepath> Use the <filepath> element to mark up both file paths and file names.    
menu cascade <menucascade> Use the <menucascade> to present a series of menu choices. The <menucascade> element contains two or more <uicontrol> elements. On output, the content of the <uicontrol> elements will be bold and separated by an angle bracket (>).    
window title <wintitle> Use the <wintitle> element to mark up the title of a window, dialog box, tab, or wizard page name.    

Use the table below to select the appropriate inline element when documenting a user interface.

Interface object DITA inline element to use
Window or dialog box name <wintitle>
Icon names <uicontrol>; to include the actual icon, use the <image> element.
Screen buttons <uicontrol>
Field names, radio or checkbox names, dropdown menu selections <uicontrol>
Menu paths <menucascade>
File paths <filepath>
System messages or output of any kind <systemoutput>
Anything that the user must type verbatim <userinput>
Variables that the user must replace with their own information <varname>
